# 🐾 PetRescue

A responsive, multi-page web application designed to connect stray, abandoned, or injured animals with loving homes, volunteers, and shelter support. The platform streamlines pet adoption, volunteer coordination, and community donations through a clean, modern user interface.

---

## 🌐 Live Demo

* **Live Website:** [PetRescue on GitHub Pages](https://ritikabudhiraja2006.github.io/PetRescue/)

---

## ✨ Key Features & Functionality

* **📌 Sticky Navigation:** Top navigation bar pinned across pages (`position: sticky`) for effortless scrolling and quick menu access.
* **📱 Responsive Design:** Styled using CSS Flexbox and CSS Grid to ensure an optimal user experience across desktops, tablets, and mobile screens without sacrificing navigation clarity.
* **✅ Smart Form Validation (JavaScript):**
  * Ensures complete data entry across required fields.
  * Strict pattern verification for contact numbers (requires exactly 10 digits).
* **✨ Dynamic UI Micro-Interactions:** Custom CSS `:hover` animations and transition effects on buttons and interactive components.
* **ℹ️ About Us & Pillars of Work:** Clear visual section detailing core operations—Monetary Support, Volunteering, Fostering, and Adoption.
* **🙋‍♀️ Volunteer Portal:** Structured registration page for community members to sign up for logistics, animal care, and event support.

---

## 🛠️ Tech Stack

* **HTML5:** Semantic markup structure.
* **CSS3:** Flexbox, CSS Grid, custom media queries, keyframe animations, and transitions.
* **JavaScript (ES6):** Client-side form validation and DOM manipulation.
* **GitHub Pages:** Automated deployment and hosting.

## 💻 How to Run Locally

1. **Clone the repository:**
   `git clone https://github.com/ritikabudhiraja2006/PetRescue.git`

2. **Navigate into the project directory:**
   `cd PetRescue`

3. **Open in browser:**
   * Double-click `index.html` to open it in your web browser.
   * Or open the folder in **VS Code** and click **Go Live**.
